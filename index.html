RAG 文件問答系統

這版不需要 Hugging Face、不需要 app.py，也不需要後端。使用者在瀏覽器自行輸入 Groq API Key，上傳 PDF / DOCX / TXT 文件後，頁面會在本機端解析、切段、檢索，再把相關片段送到 Groq 產生回答。

安全邊界

API Key 不會寫進 HTML，也不會存到 GitHub。這個頁面只在使用者按下提問時，從瀏覽器直接送到 Groq API。

但它仍然是「前端直連 API」：適合個人作品展示、教學與自用。正式產品仍建議用後端代理保護金鑰。
